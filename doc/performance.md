## 优化关键点

### 1. LLM 调用合并

- 之前：每个简历区块单独调用 LLM。
- 之后：单次调用生成所有区块。
- 收益：显著减少等待时间并增强上下文一致性。

### 2. 后端异步化

- 之前：同步路由阻塞事件循环。
- 之后：`async def` + `asyncio.to_thread`。

### 3. 重试机制优化

- 之前：401 错误被当作可重试错误。
- 之后：401/400/403/404 立即抛出。

### 4. 编辑器保真和体积优化

- 之前：TipTap + CSS scoping，编辑态样式不完整。
- 之后：iframe 加载完整 HTML，样式 100% 保真。

## 2026-07-10 本地补充

- Modern Blue 默认行距和模块间距已压缩，降低超两页概率。
- 编辑器支持手动调节行距和模块间距。
- 保存编辑版时保留完整 HTML，PDF 生成会应用调节后的版式参数。
