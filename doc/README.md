## 项目概况

- 项目名称：不平 / Buping Job Seeker Assistant
- Slogan：人生之路总是坎坷，这也造就了我们不平凡的人生
- 技术栈：React 19 + Vite 6 + FastAPI + LangChain
- 核心功能：简历生成、面试准备、模拟面试、自动化投递
- 近期重点：iframe WYSIWYG 编辑、实时预览、文档上传解析、简历排版优化
